# Mockup Icon Assets

Assets extracted as reusable SVG approximations from the current TB-IA interface mockup.
These files are intentionally not wired into the React interface yet.

Main groups:
- Brand: tbia-symbol, tbia-lockup, sus-symbol.
- Sidebar navigation: nav-territorial-analysis, nav-care-followup, nav-public-data, nav-alerts, nav-reports, nav-health-territories, nav-user-management, nav-settings, nav-help.
- Header/actions: menu, action-filter, action-refresh, action-export, action-expand, action-close.
- Map/status: map-zoom-in, map-zoom-out, map-selected-municipality, status-risk-high, status-risk-moderate, status-risk-low.
- Product feature cards: feature-territorial-vision, feature-care-management, feature-responsive-experience.
- Data panels: readiness-bars, priority-ranking.

Most icons use currentColor and can be used inline, as masks, or converted into React components.
Brand and feature icons include fixed colors matching the mockup palette.
